JOGO DA MEMÓRIA



-----INSTRUÇÕES PARA BAIXAR E EXECUTAR O JOGO-----

Baixe o  arquivo Memory_Game_Udacity.zip.
Descompacte o arquivo em qualquer local desejado e abra o arquivo udacity-memory-game.html.


---DEPENDÊNCIAS-----
O jogo não requer nenhuma biblioteca específica para rodar, apenas um browser.

-----COMO JOGAR-----

Vire as cartas e descubra os pares iguais. O jogo acaba quando todos os pares foram encontrados.
Termine o jogo com o menor número de erros possíveis para aumentar sua classificação.
Para recomeçar, clique no botão de atualização do lado direito da classificação.

-----CLASSIFICAÇÃO-----

O jogador começa o jogo com 3 estrelas no total. Ao lado das estrelas, há um contador de movimentos errados.
A cada 5 movimentos errados, é descontado uma estrela.
A menor classificação possível é de uma 1 estrela.
